github+hexo+hexo-admin搭建个人免费博客

  • 2019-12-01

  1. hexo 安装位置： D:\hexo\hexospace
    hexo 访问地址： http://localhost:4000
    hexo 配置指南： https://hexo.io/zh-cn/docs/configuration.html

  2. hexo 基本操作

    $ npm install -g hexo #安装 

$ cd /f/Workspaces/hexo/
$ hexo init #初始化


修改hexo 配置
hexo 配置指南： https://hexo.io/zh-cn/docs/configuration.html

$ hexo g # 生成
$ hexo s # 启动服务

hexo 访问地址：  http://localhost:4000

3.修改主题

hexo主题官网：https://hexo.io/themes/

下载主题
$ cd /f/Workspaces/hexo/
$ git clone https://github.com/litten/hexo-theme-yilia.git themes/miho

修改配置

文件：
_config.yml
位置：
theme: miho

命令：
hexo clean #清理
hexo g #生成

4.上传到github

修改 _config.yml ：
deploy:
  type:git
  repository:git@github.com:dervish0927/blog.git
  branch:master

三步走：

hexo clean #清理
hexo g #生成
hexo d # 发布

如果hexo d 执行后没有任何提示信息，说明deploy配置少了冒号后面少了空格！
deploy:
  type: git
  repository: git@github.com:dervish0927/blog.git
  branch: master

5.hexo 插件 hexo-admin 

安装： npm install --save hexo-admin

启动：hexo server -d

访问： http://localhost:4000/admin/ 

settings添加用户名密码，deploy运行脚本

通过hexo admin，轻松在线编辑MD文件  ----> 要求掌握.MD 语法
